Creemos un ficheiro de vídeo

Una vez consideres que todo está listo para publicar a túa obra de arte, é o momento de crear o ficheiro de vídeo. Para elo, debes entrar no Módulo de reprodución e premer sobre o botón Exportar.

Fig. 60 Opción Exportar no Modulo de reprodución

Terás un cadro de diálogo que axudarache durante todo o proceso de exportación. O primeiro que debes escoller, é o formato no que queres o vídeo da túa animación:

Fig. 61 Diálogo de selección de formato

A seguir, debes seleccionar as escenas que queres exportar. Neste caso, tratase só dunha:

Fig. 62 Diálogo de selección de escenas

Finalmente, selecciona o nome do ficheiro, a ruta onde queres exportalo e se o desexas, podes cambiar a dimensión á que queres que se cree o vídeo:

Fig. 63 Diálogo de creación do ficheiro

Una vez que premas o botón Exportar, o teu vídeo estará listo para ser visto por todo o mundo! Se o desexas, podes publicalo na nosa galería de proxectos, só tes que escribirnos comentando o teu desexo de facelo público, seguindo as nosas instrucións.

Fig. 64 Vídeo final

You can view the source file of this project, which you can find aquí así como o ficheiro de vídeo listo para ver. Non se trata dunha obra de arte revolucionaria, mais agardamos que che motive a crear grandes proxectos!
