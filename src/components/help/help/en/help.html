Help Module

If you’re reading this paragraph, then you’ve found the Help Module. This is the third tab of the workspace and provides a small manual in which we hope to give you the information you need to start enjoying this program and creating your own animations.

On the right side of the workspace, you’ll see the table of contents. To go to a section or particular topic, just to click on the item that interests you.

Fig # 29. Help Module Interface

Tip: Whenever you wish to consult the help module, simply press Ctrl + 3.

Fig # 30. Help Mode Direct Access

Remember that if you have questions or concerns that are not covered in this manual, you can seek further help from the project website http://www.maefloresta.com where you can interact with other users and the Tupi. development team. Together as a community, with your contributions, ideas and suggestions, we can turn this project into a professional tool that allows free birth to millions of pieces of animated art. But one step at a time, it’s time to make an example.
