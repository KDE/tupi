Let’s create a video file!

Once you consider that everything is ready to publish your artwork, it’s time to create the video file. This requires that you to enter the animation module and click on the Export button.

Fig #60. Export option in the animation module

A sequence of dialog boxes will take you through the options for the export process. The first thing is to choose the format you want for video of your animation:

Fig #61. Format Selection Dialog

Then you select the scenes you want to export. In this case, there is only one:

Fig #62. Scene Selection Dialog

Finally, select the path and file name, for the exported video, and if you want you can also change the dimensions and frame rate of the video that will be created:

Fig #63. Dialog File Creation

Once you have clicked the Export button, your video is ready to be seen by everyone! If you want you can post it on our project gallery, just write expressing your wish to go public, and then follow our instructions.

Fig #64. Final video

You can view the source file of this project, which you can find here and also the exported video file which is ready for viewing. This is not a revolutionary work of art, but we hope that it motivates you to create great projects!
