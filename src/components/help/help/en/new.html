Let’s start a project

Okay,first it’s time to open Tupi. Whichever graphical environment you are working with, if you already have the program installed on your computer, you should look in the section "Graphics" in your main application menu:

Fig # 32. Opening Tupi from Ubuntu

Once open, you’ll see a screen like this on your computer:

Fig # 33. Home Screen

At this point, you have two options: Create a new project or open an existing one. As this is our first time, we will create a new project by clicking on the “New Project” icon.

Fig # 34. Creating a new project

On the dialog box that appears, you can enter some basic information about the project; file name, author’s name, the dimensions we want to give our animation and the number of frames per second (FPS) with which to work initially.
For this example, we will simply fill in the first two fields and leave others with their default values.

Fig # 35. Configuration dialog for a new project

Clicking the OK button, makes the workspace available to start our project. Our starting point is always the Animation module and more specifically, the first frame of animation. Okay, we’re ready to start drawing!
